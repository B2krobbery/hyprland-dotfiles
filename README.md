## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/B2krobbery/hyprland-dotfiles
```

Enter the directory:

```bash
cd hyprland-dotfiles
```

Make the installer executable:

```bash
chmod +x install.sh
```

Run the installer:

```bash
./install.sh
```
